Installation

Installation de l'éditeur​

Afin de pouvoir utiliser l'éditeur de Practice, vous dévrez exécuter le fichier PracticeEditorInstaller.exe fourni par Altheria. Dans la plupart des cas Microsoft Defender SmartScreen bloquera l'installeur parce qu'il n'est pas reconnu par la base de données de Microsoft.

Afin de continuer l'installation, appuyez sur "Plus d'informations" et "Exécuter quand même".

Au démarrage de l'installeur vous pouvez choisir la langue (NB: cette langue ne s'applique qu'à l'installeur, pas à Practice).

Au démarrage de l’installeur sont présentés les termes de la licence. Ceux-ci disposent d’une version complète se trouvant dans l’EULA signé par votre organisation.

U kan de locatie van de software aanpassen.

La création d’une icône sur le bureau placera celle-ci sur le bureau de tous les utilisateurs du poste de travail.

Une fois les options sélectionnées, vous pouvez lancer l’installation.

L’installation peut prendre plusieurs minutes.

L’installation est maintenant finie.

Vous pouvez décider de lancer l’application ou de fermer simplement la fenêtre.

Paramétrage du casque VR​

Practice est entièrement compatible avec le standard OpenXR1, vous pourrez donc y utiliser une grande partie des casques de réalité virtuelle se trouvant dans le marché.

Afin de connecter le casque à Practice, il faudra utiliser un intermédiaire aussi appelé runtime. Il en existe deux principaux sur Windows: SteamVR2 et Oculus3, mais le choix de runtime dépend du casque à votre disposition. Nous vous conseillons d'utiliser le runtime Oculus pour les casques développés par Meta et SteamVR pour le reste.

Oculus​

Le runtime Oculus est compatible avec les casques Meta (Rift et toutes les variantes Quest). Les casques Quest devront être connectés à l'ordinateur par cable Link.

Pour plus d'information par rapport au paramétrage de votre casque Oculus, nous vous conseillons de suivre les instructions officielles de Meta.

SteamVR​

Le runtime SteamVR doit être installé à travers de la plateforme de distribution Steam4. Steam nécessite la création d'un compte pour utiliser ses services.

La procédure d'installation est similaire à tous les appareils est similaire, à part quelques différences de paramétrage spécifiques au hardware. Une liste non-exhaustive pour paramétrer les appareils avec SteamVR:

  • Valve Index: guide.
  • HTC Vive: guide.

Autres​

D'autres casques connectés existent sur le marché avec des applications propriétaires, tels que Varjo5 ou Pico6. Par contre, ces applications se servent d'un runtime comme SteamVR pour communiquer avec des applications VR.

Une fois l'installation finie, et le casque initialisé, passez à la séction Gestion de votre projet.
